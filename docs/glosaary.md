# Terminology Glossary

| Term | Meaning |
|---|---|
| PSSR | Pre Safety Startup Review |
| PSSR Initiator | User responsible for creating and managing PSSR |
| Area Owner | User responsible for final approval |
| Annexure | Checklist/document section attached to PSSR |
| Punch List | List of pending observations/tasks |
| MOC | Management of Change |
| Non-MOC | Independent PSSR workflow not linked to MOC |
